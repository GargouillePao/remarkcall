# RemarkCall
---
这是一个实时应用，现在具有实时编辑文档，实时评论和聊天三个功能。

开发工具总览：

- 前端
  - ui：bootstrap
  - 富文本编辑器：[summernote](https://github.com/summernote/summernote/)
  - 传输处理：[diffDOM](https://github.com/fiduswriter/diffDOM)
- 后台(nodejs)
  - 基础框架：express
  - 实时通信：socket.io
